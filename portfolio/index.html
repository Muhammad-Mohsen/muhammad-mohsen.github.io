about 
					I've been coding for about 15 years now, and I've loved -almost- every minute of it.
					From building enterprise-level Angular, Android, and Flutter apps to diving into the creative world of UI design and game development, I've had the chance to work on some truly exciting projects.
					During Covid, I spent a lot of time learning about UI design; I wanted to take my hobbyist-level skills to the next level. I took the 100 days of UI challenge and learned a lot from it. This allowed me to do some design work on the clock.
					I love the challenge of designing user interfaces that are functional, accessible and visually appealing.
					When I'm not coding or designing, I usually do some 3D Industrial design...The standard fare of supercars/jets/guns, nothing too highbrow!
					

					

					I'm always up for a new challenge, so if you have a project in mind, don't hesitate to reach out. Let's create something amazing together!
